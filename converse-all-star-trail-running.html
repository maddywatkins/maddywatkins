Converse All Star x Trail Running

A performance trail reinterpretation of an iconic court silhouette, exploring how heritage design language can be adapted for grip, durability, protection, and movement across uneven terrain.

BriefReimagine a classic sneaker for trail running
FocusTraction, protection, durability, and visual identity
StatusIn progress

This project explores how the Converse All Star could evolve into a trail running shoe while keeping recognizable elements of the original silhouette.

The concept balances heritage styling with performance needs including outsole grip, upper durability, toe protection, and stability for mixed terrain.
